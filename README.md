## Development

Clone the project

```bash
git clone https://github.com/aelew/lookup.tools.git
```

Go to the project directory

```bash
cd lookup.tools
```

Install dependencies

```bash
bun i
```

Start the local development server on http://localhost:3000

```bash
bun dev
```
